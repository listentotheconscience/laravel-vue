# How To

### Run backend
```shell
$ docker-compose -f docker-compose-development.yml up -d
```

### Run frontend

```shell
$ npm i
$ npm run dev
```

# Requirements

### Backend

- Laravel 8
- PHP 7.3 or 8.0+
- PostgreSQL (latest version)

### Frontend

- ViteJs
- VueJs 3
- Axios
- Tailwind Css
- TypeScript
